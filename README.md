# 悉尼繁忙指数分析
本项目包含用于计算悉尼大都市地区繁忙指数的分析代码及相关文件。

## 项目简介
本项目通过整合多个数据集（例如：人口、企业、公共交通等），对悉尼地区的350多个统计区域（SA2）进行繁忙指数的计算和分析。同时使用空间数据和可视化技术，展示关键洞察和结果。
